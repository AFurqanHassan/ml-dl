# Module 4: Computer Vision & NLP

Applying Deep Learning to real-world data like images and text, with a strong focus on Geospatial applications.

## 🗺️ Roadmap

This module is organized into two main tracks:

### Track A: Sequence Modeling & NLP

1. **[Word Embeddings](./word_embeddings.ipynb)**: Learn how we represent words as vectors.
2. **[Sentiment Analysis (LSTM)](./sentiment_analysis_lstm.ipynb)**: Using RNNs/LSTMs to understand text emotion.
3. **[Geospatial Sentiment](./geospatial_emoji_sentiment.ipynb)**: Advanced sentiment analysis with comprehensive geospatial visualization.
4. **[Transformers Intro](./transformers_intro.ipynb)**: Understanding 'Attention' and Transformer architecture.
5. **[Hugging Face Pipelines](./huggingface_pipelines.ipynb)**: Hands-on with modern pre-trained models.

### Track B: Computer Vision

1. **[Transfer Learning (Flowers-102)](./transfer_learning_flowers.ipynb)**: Leveraging pre-trained ResNet for specific image tasks.
2. **[Object Detection (YOLOv8)](./object_detection_yolo.ipynb)**: Identifying and localizing objects using YOLOv8.

## Topics Covered:

- Convolutional Neural Networks (CNNs) and Object Detection (YOLO).
- Sequence Modeling with RNNs, LSTMs, and GRUs.
- Word Embeddings (Word2Vec, GloVe).
- Transformers & Hugging Face (Self-Attention, Pipelines).
- Data Augmentation techniques (torchvision transforms v2).
- Geospatial visualization with geopandas and matplotlib.
