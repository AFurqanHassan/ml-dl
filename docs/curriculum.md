# Curriculum Overview

This document provides a logical roadmap of the ML to LLMs learning journey.

## Module 1: Basic Machine Learning

- **Focus**: Introduction to Python for ML and basic regression.
- **Topics**: Linear Regression, Logistic Regression, Exploratory Data Analysis.

## Module 2: Advanced Machine Learning

- **Focus**: Ensemble methods and Geospatial induction.
- **Topics**: Random Forests, XGBoost, Spectral Indices.

## Module 3: Deep Learning Foundations

- **Focus**: Building your own neural networks.
- **Topics**: Perceptrons, MLPs, PyTorch, EuroSAT Classification.

## Module 4: Computer Vision & NLP

- **Focus**: Advanced architectures for images and text.
- **Topics**: Transformers, Word Embeddings, Sentiment Analysis, YOLOv8 Object Detection.

## Module 5: LLMs & Generative AI

- **Focus**: The cutting edge of AI.
- **Topics**: Prompt Engineering, RAG (Retrieval Augmented Generation), Fine-Tuning with LoRA, Building Chatbots.
