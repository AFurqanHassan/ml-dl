# Module 1: Basic Machine Learning

This module covers the core concepts of supervised learning, focusing on regression and classification problems.

## Learning Objectives

- Understand the machine learning workflow (Data loading, Exploratory Data Analysis, Preprocessing, Modeling, Evaluation).
- Implement Linear Regression for predicting continuous values.
- Implement Logistic Regression for binary classification.
- Understand basic evaluation metrics like Mean Squared Error (MSE) and Accuracy.

## Prerequisites

- Basic Python knowledge.
- Familiarity with libraries like `pandas`, `numpy`, and `matplotlib`.

## Hands-on Exercises

1. **[Linear Regression](./linear_regression.ipynb):** Predicting house prices using the California Housing dataset (includes spatial coordinates).
2. **[Logistic Regression](./logistic_regression.ipynb):** Predicting survival on the Titanic.
3. **[Spatial Basics](./spatial_basics.ipynb):** Introduction to GeoPandas and spatial visualizations.
